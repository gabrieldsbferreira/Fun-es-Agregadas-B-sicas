5 queries SQL do LeetCode.  
1. Not Boring Movie: id ímpar, não chato, ordena por rating.  
2. Average Price: média ponderada por período, LEFT JOIN + 0 se sem venda.  
3. Project Employees: média de experiência por projeto.  
4. % Users Contest: inscritos únicos / total * 100.  
5. Queries Quality: AVG(rating/position) e % de rating < 3.
